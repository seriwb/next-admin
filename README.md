# admin application template

Admin application template using Next.js 16 (App Router) with yarn workspaces monorepo structure.

## base libraries
- Next.js v16
- Better Auth
- TailwindCSS
- prisma
- yarn workspace
- quill
- resend

### infrastructures
- docker compose
- PostgreSQL
- mailhog
- localstack
  - S3
  - SES
